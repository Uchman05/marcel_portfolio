# Portfolio Website

A professional portfolio website for a full-stack developer built with modern HTML5 and CSS3.

## Features

- **Modern Design**: Clean, minimalist design with smooth transitions and hover effects.
- **Responsive Layout**: Fully responsive design that works seamlessly on desktop, tablet, and mobile devices.
- **Smooth Scrolling**: Animated scrolling between sections for a better user experience.
- **Sticky Navigation**: Navigation bar stays fixed at the top while scrolling.
- **Semantic HTML5**: Proper use of semantic tags for better SEO and accessibility.
- **Modular CSS**: Well-organized CSS with comments and clear structure.

## Technologies Used

- **HTML5**: For the structure of the web page.
- **CSS3**: For styling and layout.

## Local Setup

To run this project locally, simply open the `index.html` file in your web browser.

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd personal_css_project
   ```

2. Open `index.html` in your browser:
   - Right-click on `index.html` and select "Open with" -> "Your Browser".
   - Or simply double-click the file.

## CSS Best Practices Used

- **CSS Variables**: Using CSS custom properties for easy theme management.
- **Box-Sizing**: `box-sizing: border-box;` for consistent element sizing.
- **Flexbox & Grid**: Used for modern, responsive layouts.
- **Media Queries**: Mobile-first approach to responsive design.
- **Smooth Transitions**: Added `transition` properties for smooth hover effects.
- **Semantic HTML**: Proper use of `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`, etc.
